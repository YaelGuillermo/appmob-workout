# 📱 Fitness Trainer Mobile App

Mobile application for a fitness platform focused on hiring personal trainers and tracking daily physical activity.
Built to work seamlessly with the **Fitness Trainer API** backend.

---

## 📌 Project Overview

This project provides the **mobile frontend application** for a fitness platform that allows users to:

* 🔎 Browse and hire personal trainers
* 📅 Schedule training sessions
* 📊 Track daily physical activity
* 📝 Log workouts and fitness progress
* 👤 Manage personal profiles

The app connects directly to the backend API to deliver a complete mobile fitness experience.
